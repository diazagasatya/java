#READ ME

This project is about creating a contact list where user can store up to 1000
contacts in the program and saving it into the disk. This program will automa-
tically saves the contacts after the user exits the program. It will also auto-
matically load the file that was created if the user succesfully exits the 
program. This is done with the help of my other classmates.
